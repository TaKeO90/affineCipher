# Affine cipher implementation in go


## Quick Start

```sh
$ make 
$ ./affineCipher -a <num> -b <num> -word <word here> -enc  # to encrypt
$ ./affineCipher -a <num> -b <num> -word <word here>   # to decrypt
```

## Example

```sh
$ ./affineCipher -a 7 -b 2 -word go -enc
$ sw 

$ ./affineCipher -a 7 -b 2 -word sw
$ go
```
